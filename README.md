# Pest and Disease Detection and Crop Health Monitoring

## Project Overview

This project focuses on analyzing agricultural datasets to identify crop health conditions, pest severity, and farmer behavior using Python.

The project was developed as part of the GEN AI & Data Science Level 1 Specialization.

- NumPy

- Seaborn
- Scikit-learn
- OpenPyXL
- Visual Studio Code
